describe

Recursively traverses an object’s properties, including along its prototype chain. Handles cycles.

Since: 0.1.0
Parameters

ObjectDescription

Type: Object

Since: 0.1.0
Properties

summary (string) : a short summary of a potentially complicated object hierarchy
isIterable (boolean?) : whether an object, all the way along its prototype chain, is iterable
properties ((Array<ObjectDescription> | Array<PrimitiveDescription>)) : all of the instance’s own propties and symbols

PropertyDescription

Type: Object

Since: 0.1.0
Properties
name (string) : the name of the property
enumerable (boolean) : whether it’s enumerable, as defined in Object.defineProperty()
configurable (string) : whether it’s configurable, as defined in Object.defineProperty()
from (string) : the type name of the object on which it was defined
is ((ObjectDescription | FunctionDescription | string)) : the name of the object’s type
value (string) : the serialized value
isPrimitive (boolean) : true
Example
{
  "name": "str",
  "enumerable": true,
  "configurable": true,
  "from": "Object",
  "is": "string",
  "value": "\"string\"",
  "isPrimitive": true
},
